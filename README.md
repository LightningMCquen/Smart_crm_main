# Smart CRM Main

## Overview

This is a Smart CRM project built with React, Vite, and Tailwind CSS. 

## Getting Started

### Installation
Ensure you have Node.js installed, then run:
```bash
npm install
```

### Running in Development
To start the Vite development server:
```bash
npm run dev
```

### Building for Production
To build the application for production:
```bash
npm run build
```

The output will be generated in the `dist` directory. To preview it:
```bash
npm run preview
```
